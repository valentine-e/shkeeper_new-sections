REQUIREMENTS

Server requirements for integration

To run SHKeeper with Bitcoin, Litecoin, Dogecoin, and Monero, you need a VPS or a dedicated server with 8 CPU cores, 8GB RAM, 100GB SSD.
To use Ethereum ERC20 tokens with SHKeeper, you need to have your own Ethereum Fullnode, which requires additional 4 CPU cores, 16GB of RAM, and 1TB SSD (2TB recommended).

To use Tron TRC20 tokens with SHKeeper, you have 2 options:

  • install your own Tron Fullnode; this requires additional 8 CPU cores, 32GB of RAM, and 2TB SSD (4TB recommended);

  • use our Tron Fullnode server (no additional resource required).

INTEGRATION

How could SHKeeper be integrated?

SHKeeper equips you with extra efficiency in integrating crypto payment services with any other existing systems.
Integrate with popular CMS & plug-ins

SHKeeper is available for integration with any of the most popular CMSs, plugins, and billing systems. This is a completely digitalized and decentralized option. So you don't need to rely on third parties. The module for WordPress CMS & WooCommerce and the module for WHMCS are available for free.

Integrate with your own code or product

SHKeeper is applicable to any product or code to accept cryptocurrency payments. It's an open-source crypto-processor and is available to everyone without the need for payment, and does not charge a transaction fee. You can also contact us for help with installation, configuration, or maintenance.

Use as your own wallet

It is quite possible to use SHKeeper as a regular multi-currency crypto wallet. It can be accessed at any time using your private key. It allows you to back up the blockchain key, whereas you can also restore the key through your other crypto wallets.
